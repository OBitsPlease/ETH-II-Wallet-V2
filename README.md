## What's Included

The installer bundles everything you need:

- **ETH II Wallet** — GUI wallet for sending, receiving, and viewing your ETHII balance
- **ethii.exe** — The ETH II node binary (Ethash PoW, Chain ID 20482)
- **Node auto-start** — The wallet automatically starts and syncs the local node on launch

---

## Installation

1. Download the installer from the [Releases](https://github.com/OBitsPlease/ETH-II-Wallet-V2/releases) page
2. Run `ETH II Wallet Setup v2.1.9.exe`
3. Follow the on-screen installer steps
4. Launch **ETH II Wallet** from your desktop shortcut
5. The wallet will create a new address on first launch, or you can import an existing keystore

## Notes

- This wallet runs a **local ETH II node** in the background for full trustless verification
- Your keystore file is stored in `%APPDATA%\ETH II Wallet\` — **back it up**
- The wallet connects to bootstrap peers automatically — no manual configuration needed
- CPU mining is **disabled** — ETH II is ASIC/GPU only (Ethash PoW)
